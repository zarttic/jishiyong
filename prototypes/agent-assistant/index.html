及时用 UI 原型

冰箱门上的保鲜墙

这版原型把应用从普通清单改成一面保鲜墙：物品是保鲜标签，小用是会贴到上下文里的智能便签，统计变成少浪费报告。

冷雾面板：更轻、更干净，适合让标签和今日看板成为绝对主角。

应用图标

绿色方块分别替换成分类符号，保留暖黄色小用浮层，方便比较哪个分类更适合作为主图标。

首页采用冷雾保鲜面，不画对开门中缝，用贴纸、磁吸标签、分层收纳制造记忆点。

小用不做人形头像，它是一张能移动、能展开、能确认操作的便签。
